# 개발 카드 이미지 폴더

여기에 이미지를 넣으면 해당 카드의 그림으로 사용됩니다. **없는 카드는 자동으로 도트(픽셀) 일러스트로 표시**됩니다.

## 파일 이름 규칙
파일명 = **카드 ID** + 확장자. 예) `L1-001.png`, `L2-015.jpg`, `L3-007.webp`

지원 확장자: `.png` `.jpg` `.jpeg` `.webp`

## 카드 ID 범위 (총 90장)
- 레벨 1: `L1-001` ~ `L1-040`
- 레벨 2: `L2-001` ~ `L2-030`
- 레벨 3: `L3-001` ~ `L3-020`

정확한 각 카드의 색/비용/점수는 `data/cards.json`에서 확인할 수 있습니다.
(일부만 넣어도 됩니다 — 넣은 것만 이미지로 바뀌고 나머지는 도트로 유지)

## 권장 사항
- 카드 중앙 "그림 영역"에 채워집니다(object-cover). 가로형(예: 4:3) 또는 정사각형 권장.
- 권장 크기: 가로 약 256~512px.

## 적용 방법
파일을 넣은 뒤:
- 개발 중이면 `npm run assets` 실행(또는 dev 서버 재시작) → 매니페스트 갱신
- 배포는 빌드 시 자동 반영(`prebuild`에서 스캔)

> 매니페스트(`lib/assets.ts`)는 자동 생성 파일이라 직접 수정하지 마세요.
