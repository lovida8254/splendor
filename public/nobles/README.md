# 귀족 타일 이미지 폴더

여기에 이미지를 넣으면 해당 귀족 타일의 초상화로 사용됩니다. **없는 귀족은 모노그램(이니셜) 문장으로 표시**됩니다.

## 파일 이름 규칙
파일명 = **귀족 ID** + 확장자. 예) `N-01.png`, `N-07.jpg`

지원 확장자: `.png` `.jpg` `.jpeg` `.webp`

## 귀족 ID 범위 (총 10장)
`N-01` ~ `N-10` (각 귀족의 이름/요구치는 `data/nobles.json` 참고)

## 권장 사항
- 타일의 초상화 영역(정사각형)에 채워집니다(object-cover). 정사각형 권장.
- 권장 크기: 약 96~256px.

## 적용 방법
파일을 넣은 뒤 `npm run assets` 실행(또는 dev 재시작). 배포는 빌드 시 자동 반영.
